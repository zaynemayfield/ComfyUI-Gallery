# Demo Video Script

Use this as a quick recording plan for a short public demo.

Target length: 90 seconds to 3 minutes.

## Setup Before Recording

- Use a folder with images, videos, and at least one compactable group.
- Include one file with useful ComfyUI metadata.
- Have at least one subfolder with media.
- Keep ComfyUI zoom/layout readable.
- Avoid showing private prompts, paths, API keys, or local network details.

## Recording Flow

### 1. Open From Toolbar

Say:

> ComfyUI Gallery opens directly from the ComfyUI toolbar, so you can browse output media without leaving the workflow.

Show:

- Toolbar button.
- Gallery opening.
- Loading state if scan is still running.

### 2. Browse Output Media

Say:

> The grid is built for large output folders, with preview sizes and incremental loading.

Show:

- Small/Medium/Large.
- 20/40/60 batch selector.
- Scroll behavior.

### 3. Filters And Search

Say:

> Search can target filenames or metadata, including prompts, model names, and seeds.

Show:

- Filename search.
- Positive Prompt or Metadata search.
- All/Images/Videos filter.
- Date range if useful.

### 4. Folder Navigation

Say:

> Folder controls let you browse just one folder or include its subfolders.

Show:

- Folder row.
- Subfolders Off/On toggle.
- Folder actions row if appropriate.

### 5. Compact Mode

Say:

> Compact mode groups related image and video outputs, including audio-enhanced variants, into one card.

Show:

- Compact Off/On.
- Compact badge.
- Left/right chevrons on card.

### 6. Preview And Metadata

Say:

> The preview has fit-to-screen media, video controls, metadata, raw metadata, downloads, and file actions.

Show:

- Open image preview.
- Metadata panel.
- Raw metadata.
- Open video preview.
- Mute/volume/loop/fullscreen controls.
- Download one file or a compacted zip.

### 7. Multi-Select

Say:

> Multi-select supports bulk download, move, and delete. In Compact mode it asks whether related files should be included too.

Show:

- Checkbox selection.
- Select All.
- Download selected.
- Move modal or Delete confirmation.
- Compact related prompt.

### 8. Close With Safety

Say:

> The fork is scoped to the ComfyUI output directory and includes path hardening for safer public use.

Show:

- Settings briefly.
- Close gallery.

## Suggested Caption

ComfyUI Gallery adds a fast media browser directly inside ComfyUI, with metadata search, video previews, compact related outputs, folder actions, and bulk cleanup.
