# Feature Guide

This guide summarizes the main user-facing features in this fork.

## Toolbar Entry

ComfyUI Gallery opens from a compact toolbar button next to other ComfyUI controls. The button remains clickable during initial media scanning, so users can open the gallery and see loading progress instead of waiting for a disabled entry point.

## Browsing

- Responsive virtualized grid for large output folders.
- Preview sizes: Small, Medium, Large.
- Batch sizes: 20, 40, 60. More media loads as the user scrolls.
- Optional date sections with Previous Day and Next Day jump buttons.
- Date range filter.
- Include Subfolders toggle:
  - On: selected folder shows direct media and descendant folder media.
  - Off: selected folder shows only direct media.

## Search And Filtering

Search scopes:

- All
- Filename
- Metadata
- Positive Prompt
- Negative Prompt
- Model
- Seed

Media filters:

- All
- Images
- Videos

## Compact Mode

Compact mode groups files that share the same base filename, including `-audio` variants. This is useful for workflows that produce an image, a video, and an audio-enhanced video for the same output.

Example group:

```text
example.mp4
example-audio.mp4
example.png
```

The grouped card shows a compact badge and chevrons for cycling through related files. Bulk move/delete asks whether to include related compacted files.

## Preview Overlay

Click a card to open a fit-to-screen preview.

Preview supports:

- Image viewing.
- Video playback with custom controls.
- Persistent video volume and mute state.
- Loop toggle.
- Fullscreen.
- Metadata panel.
- Raw metadata view.
- Delete, Move, Rename actions.

## Folder Management

The folder row supports compact browsing of root folders and subfolders. Folder actions allow:

- Create folder.
- Rename folder.
- Move folder through a tree picker.
- Delete folder with confirmation.

All folder actions are guarded by backend path checks.

## Multi-Select

Use checkboxes to enter multi-select mode.

Bulk actions:

- Delete selected.
- Move selected.
- Select all visible.
- Clear selection.

When Compact is on, bulk actions ask whether related compacted files should be included.

## Performance

- Virtualized grid.
- Lazy media batches.
- Cached video thumbnails generated by `ffmpeg`.
- Video duration metadata generated by `ffprobe` when available.
- Autoplay is off by default.
- Videos only autoplay in-grid when Autoplay is on and the card is visible.

## Safety Model

This fork is scoped to ComfyUI's output directory:

- Absolute gallery paths are rejected.
- Paths escaping the output directory are rejected.
- Static gallery route does not follow symlinks.
- File actions reuse backend path validation.

The gallery can still delete, move, and rename files inside the output directory, so users should not expose ComfyUI to untrusted networks.
